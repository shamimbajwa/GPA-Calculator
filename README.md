# 🎓 GPA Calculator

A simple and user-friendly GPA Calculator built using HTML, CSS, and JavaScript. This project helps students easily calculate their GPA based on subject grades and credit hours.

## 🚀 Features
- Add multiple subjects
- Input grades and credit hours
- Automatic GPA calculation
- Easy to use interface

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📌 How to Use
1. Enter subject names (optional)
2. Add credit hours for each subject
4. Click on **Calculate GPA**
5. Your GPA will be displayed instantly
